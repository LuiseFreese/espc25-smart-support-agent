# Comprehensive Password Recovery and Reset Guide

## Overview

This guide provides detailed instructions for resetting your corporate account password, recovering forgotten passwords, and troubleshooting common password-related issues.

## Self-Service Password Reset

### Step-by-Step Instructions

1. **Navigate to the Password Reset Portal**
   - Go to https://portal.example.com/login
   - Click "Forgot Password?" below the login form
   - Or visit https://passwordreset.example.com directly

2. **Verify Your Identity**
   - Enter your corporate email address or username
   - Complete multi-factor authentication:
     - SMS code to registered mobile number
     - Email verification to alternate email
     - Security questions (if configured)

3. **Create New Password**
   - Password must meet these requirements:
     - Minimum 12 characters
     - At least one uppercase letter (A-Z)
     - At least one lowercase letter (a-z)
     - At least one number (0-9)
     - At least one special character (!@#$%^&*)
     - Cannot reuse last 5 passwords
   - Confirm your new password by typing it twice

4. **Complete Reset**
   - Click "Reset Password"
   - Wait for confirmation message
   - You can now log in with your new password

## Common Issues and Solutions

### "Forgot Password Link Not Working"

**Symptoms:**
- Click "Forgot Password?" but nothing happens
- Error message when trying to access reset portal
- Reset link in email doesn't work

**Solutions:**
1. Clear browser cache and cookies
2. Try a different browser (Chrome, Edge, Firefox)
3. Check if you're using the correct portal URL
4. Disable browser extensions temporarily
5. Try incognito/private browsing mode

### "Not Receiving Password Reset Email"

**Symptoms:**
- Requested password reset but no email arrives
- Waiting more than 10 minutes with no response

**Solutions:**
1. Check your spam/junk folder
2. Verify you entered the correct email address
3. Check if your mailbox is full
4. Add noreply@example.com to your safe senders list
5. Wait 15 minutes and try requesting again
6. Contact IT support if still no email after 30 minutes

### "Password Reset Link Expired"

**Symptoms:**
- Click reset link from email but get "expired" error
- Link was working before but not anymore

**Solutions:**
- Reset links expire after 24 hours for security
- Request a new password reset
- Complete the reset within 24 hours of receiving the link
- Check the timestamp on the email to confirm link age

### "New Password Not Accepted"

**Symptoms:**
- Password meets requirements but still rejected
- "Password does not meet complexity requirements" error

**Solutions:**
1. Ensure password is at least 12 characters
2. Avoid common words or patterns
3. Don't use your name, username, or email
4. Include all required character types
5. Check for spaces at beginning or end
6. Try a completely different password

### "Account Locked After Reset"

**Symptoms:**
- Successfully reset password but can't log in
- "Account locked" or "too many attempts" message

**Solutions:**
1. Wait 30 minutes before trying again
2. Ensure caps lock is off
3. Type password carefully (don't copy-paste first time)
4. If still locked, contact IT support for immediate unlock
5. Account auto-unlocks after 30 minutes

## Multi-Factor Authentication (MFA) Setup

After resetting your password, you may need to reconfigure MFA:

1. Install Microsoft Authenticator app on your phone
2. Scan QR code during first login
3. Verify with one-time code
4. Register alternate phone number (recommended)
5. Save backup codes in secure location

## Password Best Practices

### Creating Strong Passwords

- Use a passphrase: "BlueElephant$Dance2025"
- Avoid personal information
- Don't reuse passwords across accounts
- Use a password manager
- Change password every 90 days

### What NOT to Do

- Don't share passwords with colleagues
- Don't write passwords on sticky notes
- Don't use same password for personal and work accounts
- Don't email passwords to yourself
- Don't save passwords in browsers on shared computers

## Security Considerations

### When to Change Your Password Immediately

- Suspect your account was compromised
- Received suspicious password reset emails you didn't request
- Logged in from public computer
- Shared password with someone
- Former colleague who knew password left company

### Signs of Compromised Account

- Emails you didn't send in Sent folder
- Unknown devices in recent login history
- Password no longer works
- Unexpected MFA requests
- Files shared with unknown people

If you notice these signs:
1. Change password immediately
2. Report to security@example.com
3. Review recent account activity
4. Check for unauthorized file access
